# Qualixar Media CDN

Public CDN for Qualixar marketing media. Serves MP4s via `raw.githubusercontent.com` for IG Reels Graph API and other tools that require unauthenticated public URLs.

## Structure
- `shorts/agentassert/` — AgentAssert ABC explainer Shorts/Reels (60s, 9:16, 1080×1920)

## Usage
Append filename to base URL:
`https://raw.githubusercontent.com/qualixar/media-cdn/main/shorts/agentassert/<filename>`
